# Have A Name

插件可以通过` (xt|dt|cl): 变量 ` 翻译为 `variable`，让你的变量拥有一个名字；

## 设置


| Name|Description|
| ------------------------- | ---------------------------------- |
| `vscodeFanyi.Apifrom`|所使用的翻译平台,目前只支持百度（BD）和有道（YD） |
| `vscodeFanyi.Apikey` | 翻译平台提供的账户秘钥 |
| `vscodeFanyi.time` | 输入后触发翻译的延迟，默认500毫秒（防止无效翻译） |

## ⚠️ 
- ( xt | dt | cl ) 不区分大小写
- 更改配置后建议重启.
